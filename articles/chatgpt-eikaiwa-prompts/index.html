ChatGPT英会話プロンプト集【コピペで使える20選2026】

本記事はアフィリエイト広告を含みます。

「ChatGPTで英会話を練習したいけど、どう入力すれば良いかわからない」そんな悩みを持つ方のために、すぐに使えるプロンプトを20個厳選してまとめました。コピペするだけで今日から英語学習を始められます。

コピペで使える英会話練習プロンプト10選

ChatGPTを英会話学習に活用する最大のハードルは、最初の一文をどう入力するかです。何となく「英語を教えて」と打っても、返ってくる回答は漠然としがちで、実際の会話練習にはなかなか結びつきません。大切なのは、ChatGPTに「役割」と「状況」と「ルール」を明確に伝えることです。

以下のプロンプトは、初心者から中級者まで幅広く使えるように設計しています。そのままコピーして貼り付け、必要な部分だけ自分の情報に書き換えてお使いください。

---

プロンプト1：フリートーク練習（初心者向け）

```

あなたはフレンドリーな英語ネイティブスピーカーです。

私はA2レベルの英語学習者です。

日常的なテーマ（趣味・天気・週末の過ごし方など）について

シンプルな英語で会話してください。

私が文法ミスをしたら、会話の流れを止めず、

返答の中に自然な形で正しい表現を織り交ぜてください。

まず最初に、あなたから話しかけてください。

```

このプロンプトのポイントは、「ミスを自然に直してほしい」というルールを事前に伝えている点です。いちいち「それは間違いです」と指摘されると会話のリズムが崩れますが、この方法なら自然な会話の中で正しい英語を吸収できます。

---

プロンプト2：自己紹介練習

```

私の英語の自己紹介を練習したいです。

あなたは初めて会うアメリカ人のビジネスパーソンを演じてください。

私が自己紹介をしたら、自然な質問を2〜3個してください。

会話は英語のみで進め、私の返答が不自然な場合は

会話終了後にまとめてフィードバックをください。

```

---

プロンプト3：音読フィードバックの代替練習

```

以下の英文を声に出して読む練習をするために、

この英文についてQ&A形式で質問してください。

英文：[ここに練習したい英文を貼り付ける]

質問は3つ、英語でしてください。

私の回答に対して、表現の良い点と改善点を日本語で教えてください。

```

---

プロンプト4：聞き返し表現の練習

```

あなたは英語ネイティブスピーカーです。

私に対して、少し速めのナチュラルな英語で話しかけてください。

私の目標は「聞き取れなかったときに自然に聞き返す表現」を

実際の会話で使えるようにすることです。

会話を通じて、聞き返し表現（Could you say that again? など）を

使う練習をサポートしてください。

```

---

プロンプト5：感情・意見を英語で伝える練習

```

あなたは英会話コーチです。

以下のテーマについて、私が英語で意見を述べます。

テーマ：「リモートワークは良いと思うか」

私の意見を聞いた後、賛成・反対どちらかの立場から

英語で反論してください。ディベート形式で会話を続けましょう。

会話後、私の英語表現で気になった点を日本語で3つ教えてください。

```

---

プロンプト6：日記英訳フィードバック

```

今日あったことを日本語で書きます。

それを英語に翻訳してください。

その後、より自然なネイティブ表現があれば提案してください。

日本語：[今日の出来事をここに書く]

```

---

プロンプト7：語彙拡張会話

```

あなたは英語の先生です。

「travel（旅行）」というテーマで私と英語で会話してください。

会話の中で、中級者が覚えておくと役立つ語彙や表現を

自然に使い、会話後にそれらをリストアップして

日本語で意味を教えてください。

```

---

プロンプト8：スピーキング原稿チェック

```

以下の英語スピーチ原稿を添削してください。

チェックポイント：

  1. 1. 文法の誤り
  2. 2. より自然な表現への言い換え
  3. 3. 全体的な流れと論理構成

添削後、改善版の全文も示してください。

原稿：[ここにスピーチ原稿を貼る]

```

プロンプト9：シャドーイング素材の作成

```

私のレベルはB1（中級）です。

「環境問題」についての英語スピーチ原稿を作成してください。

条件：

・長さは100〜120語

・難しい単語は避け、自然な話し言葉調にする

・作成後、難しい単語と表現に日本語注釈をつける

```

---

プロンプト10：間違い修正モード

```

これから私が英語で文章を書きます。

文法・語彙・自然さの観点からすべて添削し、

何がどう間違っていたかを日本語で説明してください。

説明は「なぜその表現が不自然なのか」まで踏み込んでください。

```

これら10個のプロンプトだけでも、毎日異なる形で英会話練習が可能です。同じプロンプトを繰り返し使いながら、少しずつ内容をアレンジしていくことで、継続的なスキルアップが期待できます。

シチュエーション別ロールプレイプロンプト

英語学習で挫折しやすい理由のひとつは、「勉強した表現が実際の場面で使えない」という経験です。ロールプレイはその橋渡しをしてくれる練習法で、特にChatGPTとの相性が非常に良いとされています。リアルな場面設定をすればするほど、実践的な英語力が身につきます。

以下では、日常生活からよく遭遇するシチュエーションを5つ選び、そのままロールプレイに使えるプロンプトを紹介します。

---

シチュエーション1：空港・機内での会話

```

あなたは航空会社のスタッフです。

私は日本から初めてニューヨークへ向かう乗客です。

チェックインカウンターでの会話から始めてください。

荷物の預け入れ、座席の希望、搭乗ゲートの確認など

実際の場面で起きやすいやりとりを英語で行ってください。

会話終了後、私が使えるようになると便利な空港英語フレーズを

5つリストアップしてください。

```

---

シチュエーション2：レストランでの注文

```

あなたはニューヨークのカジュアルなレストランのウェイターです。

私はメニューを見ながら注文する練習をしたいです。

アレルギーの確認、おすすめを聞く、追加注文など

実際に使われる会話を英語で進めてください。

私が詰まったときは、ヒントを日本語で小さく添えてください。

```

---

シチュエーション3：ホテルのフロントでトラブル対応

```

あなたはホテルのフロントスタッフです。

私の部屋のシャワーが故障しているという設定で

クレーム対応の英語ロールプレイをしてください。

私がスムーズに要望を伝えられるよう、

詰まったときは「こう言えますよ」とサポートしてください。

会話は英語で進め、終了後に使えたフレーズを

日本語解説つきでまとめてください。

```

---

シチュエーション4：友人との日常会話（週末の話）

```

あなたは私の英語圏の友人です。

週末に何をしたかについてカジュアルに英語で話しましょう。

スラングや口語表現も自然に使ってください。

会話の中で出てきた口語表現は、後でリストにして

「フォーマルな言い方」と「カジュアルな言い方」の

両方を日本語で教えてください。

```

---

シチュエーション5：道案内を頼む・教える

```

私は海外旅行中の日本人観光客です。

あなたはその街に住んでいる地元の人を演じてください。

私が「最寄りの地下鉄駅への行き方」を聞く場面から始めてください。

方向や距離の伝え方に使われる英語表現を

会話の中で自然に使ってもらえると助かります。

```

ロールプレイの効果を高めるコツは、「終わった後に必ずフィードバックをもらう」というルールをプロンプトに含めることです。会話だけで終わると何が良くて何が悪かったかが曖昧になりがちですが、終了後のフィードバックをセットにすることで、毎回の練習が確実な積み上げになります。また、同じシチュエーションを複数回試して、前回よりも自然に話せるようになっているかを確認する使い方もおすすめです。

文法・語彙の解説を引き出すプロンプト

英会話の上達には、「なんとなく話せる」だけでなく、文法の仕組みや語彙の使い分けを理解することが土台になります。ChatGPTは文法説明や語彙解説が非常に得意で、教科書的な説明だけでなく「なぜそうなるのか」「実際にどう使われるのか」という深い部分まで引き出せます。

以下では、文法・語彙の学習に特化したプロンプトを5つ紹介します。

---

プロンプト11：現在完了と過去形の違いを例文で学ぶ

```

現在完了形（have + 過去分詞）と過去形の違いを

日本語で説明してください。

説明の後、それぞれの使い方を示す例文を5つずつ英語で作り、

なぜそちらの時制を使うのかも一文で説明してください。

最後に、日本人が間違えやすいパターンをひとつ挙げてください。

```

---

プロンプト12：前置詞の使い分けを徹底理解

```

前置詞 in / on / at の基本的な使い分けを

場所・時間・その他の用法に分けて日本語で教えてください。

それぞれに英語の例文を3つずつ添えてください。

「なんとなく覚えている」状態から「使い分けられる」状態に

なれるような説明をお願いします。

```

---

プロンプト13：似た意味の単語の使い分け

```

以下の単語の違いを教えてください。

単語グループ：big / large / great / huge

それぞれのニュアンスの違いと、

使われやすいシチュエーションを日本語で説明してください。

例文も各単語につき2つずつ英語で示してください。

```

---

プロンプト14：自分の作文から文法を学ぶ

```

以下の英文の文法的な誤りをすべて指摘してください。

誤りの種類（時制・冠詞・前置詞など）を明記し、

なぜ間違っているのかを日本語で解説してください。

正しい文も示した上で、同じ間違いをしないための

覚え方のコツも教えてください。

英文：[自分の書いた英文を貼る]

```

---

プロンプト15：ワンランク上の語彙に言い換える

```

以下の文章に使われている基本的な単語を、

より洗練された英語表現に言い換えてください。

言い換えた後、元の表現との違いとニュアンスの差を

日本語で説明してください。

対象文：[言い換えたい英文を貼る]

```

文法や語彙の学習でChatGPTを使う際の重要なポイントは、「なぜ？」を必ず追加することです。ただ例文を出してもらうだけでは表面的な暗記にとどまりがちです。「なぜその形になるのか」「なぜその単語が選ばれるのか」という理由まで引き出すことで、応用が利く本質的な理解につながります。また、一度説明を受けた後に「それでは私がその文法を使って例文を3つ作るので、チェックしてください」と続けると、理解の定着がさらに深まります。
